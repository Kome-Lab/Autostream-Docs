# 安全に公開する

AutoStream を安全に使うための基本方針です。

## 守ること

- パスワード、トークン、配信キー、Webhook URL を GitHub に置かない
- 本番環境では HTTPS を使う
- 管理画面は必要な人だけがアクセスできるようにする
- 不要なポートを公開しない
- 秘密情報は定期的に入れ替える

## 公開してよい情報

- インストール方法
- 基本的な使い方
- 一般的なセキュリティ方針
- placeholder だけの設定例

## 公開しない情報

- 実際の secret
- 内部監査ログ
- 脆弱性の詳しい再現手順
- 実運用のスクリーンショットや検証証跡
