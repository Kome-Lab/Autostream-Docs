# 権限

AutoStream の管理画面を複数人で使う場合の考え方です。必要な人に、必要な権限だけを付けます。

## 管理者に向いている人

- 配信先や保存先を管理する人
- token や Webhook URL を設定する人
- サービスの起動状態を確認する人
- 障害時に設定変更を行う人

管理者は、配信の開始や停止だけでなく、運用に影響する設定を変更できます。必要な人数に絞ってください。

## 一般ユーザーに向いている人

- 配信状態を見る人
- 録画結果を確認する人
- 日常チェックだけを行う人

一般ユーザーには、secret や配信キーを変更できる権限を付けない方が安全です。

## 見直すタイミング

1. 新しい担当者を追加したとき
2. 担当から外れた人がいるとき
3. token や配信先を変更したとき
4. 本番公開前
5. 定期メンテナンス時

共有アカウントは避け、誰が操作したか分かる状態にしてください。

## 運用のコツ

- 最初は管理者を少なくする
- 作業担当と閲覧担当を分ける
- 退職者や担当外のアカウントを残さない
- 権限変更後はログイン確認を行う
