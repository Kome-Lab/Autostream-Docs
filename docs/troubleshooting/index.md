# 困ったとき

問題が起きたら、まず次の順番で確認してください。

1. サービスが起動しているか
2. Control Panel にログインできるか
3. 配信先や Discord の設定が正しいか
4. サーバーの時刻、ネットワーク、ファイアウォールに問題がないか
5. ログにエラーが出ていないか

## よくある原因

- トークンや配信キーの入力間違い
- サーバーから外部サービスへ接続できない
- Discord Bot に必要な権限がない
- YouTube 側の配信枠が準備できていない
- 保存先の容量や権限が足りない

## 文字化けしたとき

Windows PowerShell で日本語が崩れて見える場合があります。まずファイル自体ではなく、表示エンコードの問題を疑ってください。UTF-8 として読み直してから判断します。
