# 文字化けしたとき

日本語が崩れて見える場合の確認手順です。特に Windows PowerShell では、ファイル自体ではなく表示側の文字コードが原因になることがあります。

## まず確認すること

- ブラウザ上の表示も崩れているか
- PowerShell だけで崩れて見えるか
- ファイルが UTF-8 として保存されているか
- コピー元のテキストがすでに崩れていないか

## PowerShellで見る場合

1. `Get-Content -Encoding UTF8` で読みます。
2. ターミナルの表示フォントを確認します。
3. 可能であればエディタやブラウザでも同じファイルを開きます。
4. build 結果の HTML で正しく表示されるか確認します。

## 直らないとき

- ファイルを UTF-8 で保存し直します。
- コマンド出力ではなく、元ファイルを確認します。
- ログを共有するときは secret が含まれていないか確認します。

PowerShell だけで崩れる場合、まず表示エンコード問題として扱ってください。
