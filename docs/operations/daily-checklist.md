# 日常チェック

本番運用では、配信前と配信後に短い確認を行うとトラブルに気づきやすくなります。

## 配信前

1. Control Panel を開きます。
2. Worker と Encoder Recorder が online になっているか確認します。
3. Discord Bot が対象チャンネルに参加できるか確認します。
4. 配信先と保存先が正しいか確認します。
5. 1 分程度のテスト配信を行います。
6. 映像、音声、録画、通知を確認します。

## 配信中

- Control Panel の状態が running のままか確認します。
- 配信先のプレビューで映像と音声を確認します。
- エラー通知が来ていないか確認します。
- サーバーの空き容量が不足していないか確認します。

## 配信後

1. Control Panel から停止します。
2. 状態が stopped になったことを確認します。
3. 録画ファイルが作られているか確認します。
4. 保存先へのアップロード結果を確認します。
5. 不要な一時ファイルが増えすぎていないか確認します。

問題がある場合は、まず [困ったとき](/troubleshooting/) に戻ってください。
