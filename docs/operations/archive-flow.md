# Archive Flow

日常の使い方 に関するページです。

## まず知っておくこと

AutoStream は、配信、録画、保存、状態確認をまとめて扱うための仕組みです。最初は細かい設定をすべて理解するより、Control Panel で状態を見ながら一つずつ確認するのがおすすめです。

## 確認すること

- 必要なサービスが起動している
- 設定値に実際の secret を書き残していない
- 配信先、保存先、通知先が正しく登録されている
