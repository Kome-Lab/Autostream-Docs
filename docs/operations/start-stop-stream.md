# 配信を開始・停止する

日常運用で配信を開始、停止、確認するための手順です。

## 開始する

1. Control Panel を開きます。
2. 配信一覧から対象の配信を選びます。
3. 入力元、配信先、録画保存先を確認します。
4. 開始ボタンを押します。
5. 状態が starting から running に変わるまで待ちます。
6. 配信先のプレビューで映像と音声を確認します。

## 停止する

1. Control Panel で対象の配信を開きます。
2. 停止ボタンを押します。
3. 状態が stopping から stopped に変わるまで待ちます。
4. 録画ファイルが保存されているか確認します。
5. 必要なら保存先へのアップロード結果を確認します。

## 途中で止まったとき

- まず Control Panel の状態表示を確認します。
- 次に Worker と Encoder Recorder のログを確認します。
- 配信先側で stream key が無効になっていないか確認します。
- 録画だけ成功している場合は、配信先の設定を見直します。
- 配信は成功しているのに保存できない場合は、保存先の認証と容量を確認します。

毎日の運用では、開始前に 1 分程度のテスト配信を行うと問題に気づきやすくなります。
