# 最初の配信を始める

手順 の基本手順です。細かい内部仕様よりも、まず動かすために必要な流れだけをまとめます。

## おすすめの進め方

1. サーバーを用意する
2. env example を見ながら必要な値を設定する
3. Docker または release artifact で起動する
4. Control Panel を開く
5. Discord、配信先、保存先を登録する
6. テスト配信で音声、映像、録画、保存を確認する

## 確認ポイント

- サービスが起動している
- Control Panel にアクセスできる
- 配信先へ接続できる
- 録画ファイルが作られる
- 必要な通知が届く

## うまくいかないとき

まず [困ったとき](/troubleshooting/) を確認してください。ログを見るときは、secret が表示されていないか注意します。
