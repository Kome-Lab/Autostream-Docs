# 最初の配信を始める

インストール後、最初に短いテスト配信を行うための手順です。本番配信の前に、音声、映像、録画、通知を一通り確認します。

## 事前確認

- Control Panel にログインできる
- Worker と Encoder Recorder が online になっている
- Discord Bot が対象のチャンネルに参加できる
- 配信先の URL と stream key が登録済み
- 録画保存先に書き込みできる

## 手順

1. Control Panel で新しい配信を作ります。
2. 配信タイトル、配信先、録画保存先を選びます。
3. Discord の入力元を選びます。
4. まずは短時間のテストとして開始します。
5. 配信先のプレビューで映像と音声を確認します。
6. Control Panel で状態が running になっているか確認します。
7. 1 分程度で停止し、録画ファイルと通知を確認します。

## 本番前の確認

- 音声が小さすぎない
- 映像が止まらない
- 録画ファイルが再生できる
- 保存先へアップロードできる
- 失敗時の通知が届く

問題があれば [配信を開始・停止する](/operations/start-stop-stream) と [困ったとき](/troubleshooting/) を確認してください。
