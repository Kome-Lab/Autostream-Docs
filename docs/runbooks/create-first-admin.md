# 初回管理者を作る

Control Panel を使い始める前に、管理者アカウントを作成します。管理者は配信先、保存先、通知先、サービス状態を扱えるため、必要な人だけに付与してください。

## 手順

1. Control Panel を起動します。
2. 初回セットアップ画面、または管理者作成用の手順を開きます。
3. 管理者のメールアドレスまたはユーザー名を入力します。
4. 強いパスワードを設定します。
5. 作成後、いったんログアウトしてログインできるか確認します。
6. 不要な初期設定用 token や一時設定があれば無効化します。

## パスワードの考え方

- 他のサービスと同じパスワードを使わない
- 共有アカウントを作らない
- 退職者や担当外の人のアカウントを残さない
- 本番では HTTPS の URL からログインする

## 作成後に確認すること

- 管理画面にログインできる
- 配信先や保存先の登録画面を開ける
- 一般ユーザーに不要な権限が付いていない
- ログやスクリーンショットにパスワードが写っていない

権限の考え方は [権限](/security/roles-and-permissions) も確認してください。
